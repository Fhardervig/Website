For inline formulas, enclose the formula in $...$. For displayed formulas, use $$...$$.
These render differently. For example, type
$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$
to show $\sum_{i=0}^n i^2 = \frac{{(n^2+n)}\cssId{eq1}{(2n+1)}}{6}\class{tooltiptext}{\text{This is text}}$ (which is inline mode) or type
$$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$$
to show $$\sum_{i=0}^n i^2 = \frac{\class{tooltip term1}{(n^2+n)}{(2n+1)}}{6}\class{tooltiptext}{\text{This is text}}$$ (which is display mode2).
